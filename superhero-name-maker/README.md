# Superhero Name Maker 🦸

A JavaScript program that generates a superhero name based on user input.

## How It Works
1. User is prompted for an animal name, gender and age
2. Each input is validated using Regular Expressions (Regex)
3. Based on gender and age, a description is generated
4. Final superhero name is displayed as: [animal]-[description]

## Example
- Input: `bat`, `Male`, `25`
- Output: `The superhero name is: bat-man!`

## Concepts Used
- `prompt()` and `alert()`
- Regular Expressions (Regex) for input validation
- `parseInt()` for type conversion
- Conditionals (if/else)
- Functions
